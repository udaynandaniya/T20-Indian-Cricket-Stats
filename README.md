# T20 Indian Cricket Stats

A server-side rendered (SSR) and statically generated (SSG) web application built with **Next.js** that showcases statistics of Indian T20 cricket players.  
The project focuses on **SEO best practices**, **programmatic pages**, and **performance optimization**.
